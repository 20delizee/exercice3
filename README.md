Avant tout faite cd API puis faites npm i

Pour pouvoir acceder aux differents lien vous devrez d'abord aller sur http://localhost/3000/api/keys afin de vous créer une clé api 

Pour lancer l'api veuillez aller dans le dossier api puis faire npm run dev

Vous trouverez toute les routes sur http://localhost/3000/api-docs
